Real-Time Mandelbrot Fractal Zoomer(s)

Use left and right mouse buttons to zoom in and out.

Use middle button to pan.

I wrote the code for this back in 2003 when I was learning Java, partly based on the amazing C program XaoS..

As far as I know, for a long time, the only real-time fractal zoomer that ran in the browser is the code that I wrote. You can find the Java source here.

On Sep 17, 2011, J.B. Langston III ported XaoS to javascript. I am bit sad that I didn't get to writing this in javascript, but Langston's work is lovely and it can be found here.
